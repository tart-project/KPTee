KPTee
==

KPT is retrospective framework.  
KPTee is a front-end application for KPT.

## Get start 
https://tart-project.github.io/KPTee/

## Use local 
1. `git clone https://github.com/tart-project/KPTee.git`
1. `cd kptee`
1. `npm install`
1. `npm run build`
1. click `dist/index.html`
